### Install <a name ="install"></a>

#### Asus OS/Debian Strech <a name ="asusos"></a>

`dd if=20181023-tinker-board-linaro-stretch-alip-v2.0.8.img of=/dev/mmcblk0 bs=4MiB status=progress oflag=sync`

user: `linaro`
pass: `linaro`

camera config: `~/camera/xml`
